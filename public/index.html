BonMot Examples

With the exception of bootstrap, the examples have been kept as simple as possible for clarity and understanding.

To get the most out of these examples, you'll want to have the project open in your editor of choice. All of the code specific to the example lives in that example's directory. For the first example, it lives in public/example1.

  1. Our first example is super-simple. Two way binding with three data attributes.
  2. The second example adds a control binding to the first.
  3. The third example show how we can add functionality & reduce code complexity by leveraging BonMot's native binder.
