BonMot Examples

With the exception of bootstrap, the examples have been kept as simple as possible for clarity and understanding.

To get the most out of these examples, you'll want to follow them in-order, and to have the project open in your editor of choice. All of the code specific to the example lives in that example's directory. For the first example, it lives in public/example1.

  1. Our first example is super-simple. Two way binding with three data attributes.
  2. The second example adds a control binding to the first.
  3. Add functionality & reduce code complexity by leveraging BonMot's native binder. -
  4. Create more nicely packaged styles with bon-mot's unique attribute.
  5. Automagic Child-Views. Nested data. The basics.
  6. Child-Views - Changing Models.
  7. Child-Views - Handling nested Models with the same properties.
  8. Child-View Collections - Automagic Lists of Views & Models.
  9. Lists of Models. But with a slightly different approach to a control handler.
  10. Sorting Collections of Models with Child-Views.
  11. Low cost, complex, live attributes
