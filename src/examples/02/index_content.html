{{example_number}} Buttons are easy

Controls are bound to UI elements via clear, concise, names.

We've added a button that shows an hides the input fields. Wiring functionality was as easy as adding the class w-ctrl-toggleEdit to the button and naming the show/hide function ctrlToggleEdit. BonMot figures out the relationship and even makes the button element available via .$ctrl.toggleEdit off the view instance.

As in Example 1 If you edit the input fields, you'll see the large font name update dynamically.
