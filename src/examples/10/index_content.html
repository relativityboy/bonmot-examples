Sorting, Adding, and Removing items from lists.

In this example we move from a simple html list to a table that sorts when you click a column header. See Person.View.ctrlSortBy. Clicking on a row in the table will set that model on the Address Detail view.

We have Person.View.ctrlNewAddress and Person.View.ctrlDeleteAddress for creating and deleting models.
